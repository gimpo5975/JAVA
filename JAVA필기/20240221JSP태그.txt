1)스크립트 태그:

	◎선언(<%! %>): 이 태그는 Java에서 변수 및 메소드를 선언하는 데 사용됩니다. 선언은 일반적으로 기본 서비스 메서드 외부에 배치되며 여러 요청에서 액세스할 수 있는 클래스 수준 요소를 정의하는 데 사용됩니다.
	
	◎스크립틀릿(<% %>): 스크립트릿은 Java 코드를 JSP 페이지에 직접 포함하는 데 사용됩니다. 주로 논리를 작성하거나 계산을 수행하는 데 사용됩니다. 클라이언트가 JSP 페이지를 요청할 때마다 스크립트릿이 실행됩니다.

	◎표현문(<%= %>): 표현식문은 동적 콘텐츠를 HTML 응답으로 직접 출력하는 데 사용됩니다. 주로 HTML 콘텐츠 내에서 변수 값, 계산 결과 또는 메서드 호출 결과를 표시하는 데 사용됩니다. <%= %> 내부의 표현식이 평가되고 그 결과가 문자열로 변환되어 HTML 출력에 삽입됩니다.

2) 지시어 태그:

	◎페이지 지시문(<%@ 페이지 %>): 페이지 지시문은 현재 JSP 페이지를 처리하는 방법에 대한 지침을 JSP 컨테이너에 제공하는 데 사용됩니다. 일반적으로 JSP 파일의 시작 부분에 나타나며 콘텐츠 유형, 문자 인코딩, 오류 처리, 세션 관리 등과 같은 다양한 속성을 설정하는 데 사용됩니다.

다음은 일반적으로 사용되는 <%@ 페이지 %> 지시문 속성에 대한 간략한 설명입니다.

	◎언어: JSP 페이지에서 사용되는 스크립트 언어를 지정합니다(예: "java").

	◎contentType: JSP 페이지에서 생성된 응답의 MIME 유형을 지정합니다.

	◎pageEncoding: JSP 페이지에 사용되는 문자 인코딩을 지정합니다.

	◎추가 구성 옵션을 제공하는 import, session, errorPage 등과 같은 추가 속성이 있습니다.

요약하면, JSP 태그는 Java 코드를 포함하고 지시어를 정의하며 생성된 웹 페이지의 동작과 표현을 제어하는 ​​방법을 제공합니다. 이는 동적 및 상호 생성을 위한 강력한 메커니즘을 제공합니다.
