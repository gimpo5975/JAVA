[자바의 정석 필기]

★ 메소드란?
	1. 문장들을 묶은 것
	2. 값(입력)을 받아서 처리하고, 결과를 반환(출력)

☆ 메소드의 장점
	1. 코드의 중복을 줄일 수 있다
	2. 코드의 관리가 쉽다
	3. 코드를 재사용할 수 있다
	4. 코드가 간결해서 이해하기 쉬워진다	

★ 메소드의 작성
	1. 반복적으로 수행되는 여러 문장을 메소드로 작성
	2. 하나의 메소드는 한 가지 기능만 수해하도록 작성




[KDT 중앙정보인재개발원 필기]

★ 변수는 선언된 블록 안에서만 스택에 존재하고 블록에서 벗어나면 스택에서 제거 됨

☆ 객체 지향 프로그래밍 개발의 세 가지 단계☆
	1. 클래스를 설계한다
	2. 설계된 클래스를 가지고 객체를 생성한다
	3. 생성된 객체를 이용한다

★ 클래스
	1. 필드=(멤버변수)
	2. 생성자: 객체를 만들때 초기화
	3. 메소드(기능, 동작)

☆ 생성자 : 객체를 초기화, 기본생성자는 생략가능=> 생략하면 컴파일시 JVM이 만들어 사용
		생성자의 이름은 클래스명과 동일해야한다

★ 메소드 : 동작방식, 기능처리, 처리역할
